This project is the result of an idividual assessment for a Cert III in Game Development, where we were required to create either a 2D or 3D game using software including Unity and Visual Studio. 'VOID' is a single player and retro inspired 2D action platformer. I liked the idea of having the player explore and discover, whilst they move through an environment that has something new around every corner. The setting of 'VOID' gives me the freedom to experiment and create unique environments and scenarios for entertaining gameplay. As I continue to study and learn I plan on continuing development on 'VOID', and be publishing some of that journey here.
